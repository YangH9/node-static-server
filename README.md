# node-static-server

#### 项目介绍
简单的node静态文件服务

#### 安装教程

1. git clone git@github.com:yanghan1999/node-static-server.git

2. npm install

#### 使用说明

```
bin/anydoor # 把当前文件夹作为静态资源服务器根目录

bin/anydoor -p 8080 # 设置端口号为 8080

bin/anydoor -h localhost # 设置 host 为localhost

bin/anydoor -d /usr # 设置根目录为 /usr

```
